Módulo Base de Localización — El Salvador

Extiende los formularios de Clientes, Proveedores y Compañías con los campos fiscales obligatorios para operar legalmente en El Salvador, en cumplimiento con las disposiciones del Ministerio de Hacienda.

v17 v18 v19

¿Qué incluye este módulo?

Campos y configuraciones esenciales para la localización fiscal salvadoreña.

🏢

Datos de Compañía

Campos de NIT (Número de Identificación Tributaria) y NRC (Número de Registro de Contribuyente) integrados en el formulario de empresa.

👥

Clientes y Proveedores

Identificación fiscal local en el módulo de contactos: campos de NIT, DUI y tipo de contribuyente para personas naturales y jurídicas.

🏛️

Cumplimiento Hacienda

Estructura base lista para integrarse con los módulos de DTE (Documentos Tributarios Electrónicos) y facturación electrónica según normativa del MH.
