¿Qué incluye este módulo?

Plazos y condiciones de pago para DTE según el MH.

📅

Plazos de Pago DTE

Configuración de plazos de pago según los códigos y formatos requeridos por el Ministerio de Hacienda.

💳

Condiciones de Pago

Mapeo de condiciones de pago de Odoo a los códigos oficiales del MH para su inclusión en el DTE.

🔄

Integración Automática

Los términos de pago se incluyen automáticamente en cada DTE generado sin intervención manual.
